Part 2
1. The man command displays a manual for various commands. You can move around by using up/down and quit by pressing q
2. The l flag shows long listing for files. The a flag shows all files, even hidden ones that start with '.'
3. You can jump around by holding down ctrl and pressing left/right
4. alt+e
5. alt+a
6. alt+backspace
7. termianals are tools that can interact with shells
8. an absolute path is the full path to a file from root
9. a relative path is the path to a file from your current directory
10. a flag is an optional argument you can pass to a command to change how it acts. Examples: ls -l to list detailed results, ls -a to list hidden files, rm -r to recursively remove directories
11. -r recursively removes folders and -f will delete files as well
